TIER 1 - MELHOR OPORTUNIDADE (COMECE AQUI)
================================================================================

1. VIDRAÇARIAS AUTOMOTIVAS COM MÚLTIPLAS FILIAIS
   
   Descrição:
   - Empresas especializadas em vidro automotivo
   - 3-5 lojas em cidades diferentes
   - Faturamento: R$ 100k-300k/mês
   - Equipe: 8-15 vendedores comissionados
   
   Por quê?
   ✓ Problema específico: Caixa desorganizado por loja
   ✓ Dor: Não sabe quanto cada loja faturou
   ✓ Urgência: Alta (fatura diário, precisa rastrear)
   ✓ Fit perfeito: Múltiplas lojas + comissões automáticas
   
   Como vender:
   "Você tem 3 lojas, 3 caixas confusos. Quanto cada loja faturou hoje?
    Sistema unifica tudo em 1 dashboard. Comissões calculadas automaticamente."
   
   Contato:
   → Procure donos de vidraçarias em sua região
   → Assoc. de vidraçarias / sindicatos


TIER 1+ - OPORTUNIDADE IGUAL
================================================================================

2. FRANQUIAS DE VIDRO AUTOMOTIVO
   
   Descrição:
   - Franqueador + rede de 5-10 franqueados
   - Cada unidade é independente financeiramente
   - Faturamento por unidade: R$ 50k-150k/mês
   - Controle centralizado do franqueador
   
   Por quê?
   ✓ Problema: Franqueador não consegue auditar vendas
   ✓ Dor: Falta de padronização entre unidades
   ✓ Urgência: Crítica (precisa controlar rede)
   ✓ Fit: Sistema foi feito para múltiplas lojas
   
   Como vender:
   "Visualize em tempo real quanto cada franqueado vendeu.
    Comissões automáticas. Relatórios padronizados."
   
   Contato:
   → Franqueador (maior venda, solução enterprise)
   → Franqueados (menor venda, solução unitária)


TIER 2 - ÓTIMA OPORTUNIDADE
================================================================================

3. AUTOSHOPS COM SERVIÇO DE VIDRO + AGENDAMENTO
   
   Descrição:
   - Oficinas mecânicas com serviço de vidro automotivo
   - 1-3 lojas
   - Faturamento: R$ 80k-250k/mês
   - Clientes recorrentes (agendamentos)
   
   Por quê?
   ✓ Problema: Gerencia vendas + agendamentos (tudo misturado)
   ✓ Dor: Plano de cliente (3 trocas gratis) não é rastreado
   ✓ Urgência: Alta (receita recorrente é novo modelo)
   ✓ Fit: Sistema tem painel B2C + agendamentos
   
   Como vender:
   "Crie plano para clientes (R$ 239/ano = 3 trocas gratis).
    Eles agendam online. Sistema controla tudo."
   
   Contato:
   → Autoshops listadas no Google
   → Associações de mecânicos


TIER 2+ - BOA OPORTUNIDADE
================================================================================

4. LOJAS DE SOM/ACESSÓRIOS AUTOMOTIVOS COM MÚLTIPLAS UNIDADES
   
   Descrição:
   - Especialistas em som, acessórios, segurança para carro
   - 2-4 lojas regionais
   - Faturamento: R$ 60k-200k/mês
   - Vendedores comissionados
   
   Por quê?
   ✓ Problema: Múltiplas lojas, comissões confusas
   ✓ Dor: Equipe de vendas não confia nas contas
   ✓ Urgência: Média-Alta (equipe gera friction)
   ✓ Fit: Comissões automáticas + múltiplas lojas
   
   Como vender:
   "Cada vendedor vê sua comissão em tempo real.
    Sem reclamação. Sem retrabalho."
   
   Contato:
   → Google Maps: "som e acessórios automotivos"
   → OLX, Facebook Marketplace (procurar por lojas)


TIER 3 - OPORTUNIDADE SECUNDÁRIA
================================================================================

5. REDES DE DISTRIBUIDORAS DE AUTOPEÇAS (5-10 lojas)
   
   Descrição:
   - Distribuidora que vende para mecânicos/autoshops
   - Rede maior (5-10 lojas)
   - Faturamento: R$ 200k-500k+/mês
   - Estrutura complexa (estoque, comissões, múltiplas lojas)
   
   Por quê?
   ✓ Problema: Complexidade (múltiplas lojas, estoque, comissões)
   ✓ Dor: Não consegue saber estoque centralizado vs por loja
   ✓ Urgência: Média (já têm sistemas, mas desatualizados)
   ✓ Fit: Sistema escalável, pronto para crescer
   
   Como vender:
   "Controle estoque e vendas de todas as 10 lojas em 1 lugar.
    Comissões e relatórios automáticos."
   
   Contato:
   → Buscar distribuidoras de autopeças regionais
   → Representantes comerciais de distribuidoras
